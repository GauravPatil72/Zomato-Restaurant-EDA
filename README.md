# Zomato Restaurant Data Analysis

## Overview
Exploratory Data Analysis (EDA) on the Zomato Restaurants Global Dataset to uncover 
insights that support better business decisions around city expansion, pricing strategy, 
and customer experience.

## Objectives
- Clean and prepare the dataset for analysis
- Answer 8 business questions through visual analysis
- Extract actionable insights from each analysis

## Tools & Libraries
- Python, Pandas, NumPy
- Matplotlib, Seaborn

## Key Findings
- New Delhi dominates restaurant listings; Tier-2 cities show growth potential
- Higher price range consistently correlates with better ratings
- North Indian & Chinese cuisines lead in volume; niche cuisines lead in ratings
- Tier-2 cities offer the best value for money
- Online delivery and table booking linked to higher ratings

## Dataset
Zomato Restaurants Dataset — 21 attributes covering 9,552 restaurants across 16 countries
